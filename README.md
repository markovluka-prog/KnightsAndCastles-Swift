# Рыцари и Замки — Swift / SpriteKit

![Knights and Castles](Knights_and_Castles_1920x1080.png)

Тактическая пошаговая стратегия. Swift-версия на SpriteKit.

## Запуск

Открыть `My App.swiftpm` в Xcode или Swift Playgrounds.

## Режимы

- **2 Игрока** — локальная игра на одном экране
- **Против ИИ** — игра против тактического ИИ-противника
